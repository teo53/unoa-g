# /verify — 릴리스 전 최종 규정 준수 체크

## 체크리스트

```
┌─ 추적성 ─────────────────────────────────────┐
│ [ ] WI 링크 존재                              │
│ [ ] Slack thread 링크 존재                    │
│ [ ] PR 링크 존재                              │
└───────────────────────────────────────────────┘

┌─ Gate 산출물 ─────────────────────────────────┐
│ [ ] Security/DB gate 완료 (또는 면제 사유)     │
│ [ ] UIUX/Obs gate 완료 (또는 면제 사유)        │
│ [ ] Legal gate 완료 (또는 면제 사유)           │
│ [ ] Tax gate 완료 (또는 면제 사유)             │
└───────────────────────────────────────────────┘

┌─ 안전성 ──────────────────────────────────────┐
│ [ ] 비밀정보 커밋 없음 (.env, 키, 토큰)       │
│ [ ] 변경 범위가 "작은 PR" 원칙에 부합          │
│ [ ] /qa 실행 완료 및 결과 기록                 │
└───────────────────────────────────────────────┘

┌─ 롤백 플랜 (3줄) ────────────────────────────┐
│ 1. ...                                        │
│ 2. ...                                        │
│ 3. ...                                        │
└───────────────────────────────────────────────┘
```

## 판정

- **모두 통과** → `/ship` 진행 가능
- **1개라도 미통과** → 블로커 해결 후 재실행

# UNO A 개인정보처리방침

> 시행일: 2026-02-16 | 버전: 1.0

주식회사 UNO A(이하 "회사")는 「개인정보 보호법」 등 관련 법령에 따라 이용자의 개인정보를 보호하고, 이와 관련한 고충을 신속하고 원활하게 처리하기 위하여 다음과 같은 개인정보처리방침을 수립·공개합니다.

## 제1조 (수집하는 개인정보의 항목 및 수집방법)

### 1. 필수 수집항목

| 구분 | 수집항목 |
|------|---------|
| 회원가입 | 이메일, 비밀번호, 닉네임, 생년월일, 성별 |
| 본인인증 | 이름, 생년월일, 성별, 내외국인 정보, 휴대전화번호, CI/DI |
| 결제 | 결제수단 정보(카드사, 카드번호 일부), 결제 금액, 결제 일시 |
| 크리에이터 등록 | 실명, 활동명, 프로필 사진, 자기소개 |
| 크리에이터 정산 | 은행명, 계좌번호, 예금주명, 사업자등록번호(사업자인 경우), 주민등록번호 뒷자리(원천징수 목적) |

### 2. 선택 수집항목
- 프로필 사진, 소개글, SNS 계정, 관심 아티스트

### 3. 자동 수집항목
- 서비스 이용기록, 접속 로그(IP 주소, 접속 시간), 기기 정보(OS, 브라우저), 쿠키

### 4. 수집방법
- 회원가입, 서비스 이용, 고객상담, 이벤트 참여 시 이용자가 직접 입력
- 자동 수집 도구(쿠키, 로그 수집 도구)를 통한 수집

## 제2조 (개인정보의 수집·이용 목적)
1. 회원 관리: 가입 의사 확인, 본인 확인, 연령 확인, 부정 이용 방지
2. 서비스 제공: 콘텐츠 제공, 구독 관리, DT 구매·사용 처리, 후원 처리
3. 결제 및 정산: 결제 처리, 크리에이터 정산, 세금 신고·납부
4. 마케팅 및 광고: 이벤트·프로모션 안내 (동의 시), 맞춤형 서비스 제공
5. 분쟁 처리: 민원 처리, 분쟁 조정, 법적 의무 이행
6. 서비스 개선: 통계 분석, 서비스 품질 향상

## 제3조 (개인정보의 보유 및 이용기간)

| 보존 근거 | 보존기간 | 대상 |
|----------|---------|------|
| 계약·구독 기록 | 5년 | 전자상거래법 |
| 결제·DT 거래 기록 | 5년 | 전자상거래법, 세법 |
| 소비자 불만·분쟁 기록 | 3년 | 전자상거래법 |
| 표시·광고 기록 | 6개월 | 전자상거래법 |
| 접속 로그 | 3개월 | 통신비밀보호법 |
| 동의 증빙(약관 동의 등) | 5년 | 내부 정책 |
| 탈퇴 회원 정보 | 파기 (법정 보존기간 제외) | 개인정보 보호법 |

## 제4조 (개인정보의 제3자 제공)
회사는 이용자의 동의 없이 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우는 예외로 합니다.

### 1. 동의에 의한 제공

| 항목 | 내용 |
|------|------|
| 제공받는 자 | 결제대행사(TossPayments) |
| 제공 목적 | DT 구매 결제 처리 |
| 제공 항목 | 결제 관련 정보(주문번호, 결제금액) |
| 보유기간 | 결제 완료 후 5년 |
| 거부권 | 동의를 거부할 수 있으나, 결제 서비스 이용이 제한됩니다 |

### 2. 법령에 의한 제공
- 수사기관의 적법한 요청, 법원 판결 등

## 제5조 (개인정보의 파기)
1. 보유기간 경과, 처리 목적 달성 시 지체 없이 파기합니다.
2. 전자적 파일: 복구 불가능한 방법으로 삭제
3. 종이 문서: 분쇄 또는 소각

## 제6조 (개인정보의 안전성 확보 조치)
1. 관리적 조치: 내부 관리계획 수립·시행, 직원 교육
2. 기술적 조치: 접근 통제, 암호화(비밀번호, 정산 계좌정보), 보안 프로그램 설치
3. 물리적 조치: 전산실 접근 통제

## 제7조 (이용자의 권리·의무)
1. 이용자는 언제든지 자신의 개인정보를 조회·수정·삭제·처리정지 요청할 수 있습니다.
2. 권리 행사는 서비스 내 설정 또는 고객센터를 통해 가능합니다.
3. 대리인을 통한 권리 행사 시 위임장 제출이 필요합니다.
4. 이용자는 정확한 개인정보를 제공할 의무가 있으며, 부정확한 정보로 인한 불이익에 대해 회사는 책임을 지지 않습니다.

## 제8조 (쿠키의 운용)
1. 회사는 이용자 편의를 위해 쿠키를 사용합니다.
2. 이용자는 브라우저 설정을 통해 쿠키 허용/차단을 선택할 수 있습니다.
3. 쿠키를 차단하는 경우 일부 서비스 이용에 제한이 있을 수 있습니다.

## 제9조 (동의 증빙 관리)
1. 회사는 이용자의 약관 동의 시 다음 정보를 기록·보관합니다.
   - 동의한 약관의 버전
   - 동의 시점 (타임스탬프)
   - 동의 시 접속 IP 주소
   - 동의 방식 (체크박스, 전자서명 등)
2. 동의 이력은 서비스 내 설정 > 동의 내역에서 확인할 수 있습니다.

- 개인정보분쟁조정위원회: 1833-6972 (www.kopico.go.kr)
- 개인정보침해신고센터: 118 (privacy.kisa.or.kr)
- 대검찰청 사이버수사과: 1301 (www.spo.go.kr)
- 경찰청 사이버안전국: 182 (ecrm.cyber.go.kr)

## 3. VAT/세무 처리 (총매법 기반)

### 3.1 VAT 2단계 인식 (P0.1)

| 시점 | 회계 처리 | VAT |
|------|----------|-----|
| **DT 구매** | 계약부채(선수금) 인식 | **과세 아님** (아직 공급 미완료) |
| **DT 사용** (서비스 제공) | 매출 인식 | **VAT 과세 (공급시기)** |
| **DT 환불** | 계약부채 차감 | 매출 미인식 |
| **DT 소멸** (5년) | 매출 인식 | VAT 과세 |

### 3.2 크리에이터 정산 = 광고용역비 (P0.3)

| 항목 | 내용 |
|------|------|
| **정산 성격** | 광고용역비 (회사 비용) |
| **정산 ≠** | 소비자 결제대금의 전달 |
| **회계 분류** | 판매비와관리비 → 광고선전비 |

### 3.3 크리에이터 유형별 원천징수

| 유형 | 세금 처리 | 원천징수율 | 필요 서류 |
|------|----------|----------|----------|
| 개인 (비사업자) | 기타소득 | 8.8% (소득세 8% + 지방소득세 0.8%) | 주민등록번호 |
| 개인사업자 | 사업소득 | 3.3% (소득세 3% + 지방소득세 0.3%) | 사업자등록번호 |
| 법인사업자 | 법인소득 | 원천징수 없음 (세금계산서) | 사업자등록번호, 법인등록번호 |

### 3.4 원천징수 체크리스트

**비사업자 개인 크리에이터:**
- [ ] 주민등록번호 수집 (암호화 저장)
- [ ] 기타소득 8.8% 원천징수
- [ ] 매월 원천세 신고/납부 (익월 10일까지)
- [ ] 연간 지급명세서 제출 (익년 3월)
- [ ] 크리에이터에게 원천징수영수증 발급

**사업자 크리에이터:**
- [ ] 사업자등록번호 확인 (국세청 API)
- [ ] 세금계산서 수취 또는 발행
- [ ] 부가세 처리 (수수료 매출세금계산서)

### 3.5 정산서 필수 항목

```typescript
interface PayoutStatement {
  // 기본 정보
  statementNumber: string;      // 정산서 번호
  periodStart: Date;            // 정산 기간 시작
  periodEnd: Date;              // 정산 기간 종료
  issueDate: Date;              // 발급일

  // 크리에이터 정보
  creatorName: string;
  creatorType: 'individual' | 'business';
  taxId: string;                // 주민번호 또는 사업자번호 (마스킹)

  // 수익 내역 (광고용역비 기준)
  grossRevenue: number;         // 총 수익
  platformFee: number;          // 플랫폼 수수료 (20%)
  netRevenue: number;           // 순 수익 (광고용역비)

  // 세금
  withholdingTax?: number;      // 원천징수세 (비사업자: 8.8%, 개인사업자: 3.3%)
  vat?: number;                 // 부가세 (사업자)

  // 실지급액
  payoutAmount: number;

  // 지급 정보
  bankName: string;
  accountNumber: string;        // 마스킹 (****1234)
```

## 4. 연령 제한 및 미성년자 보호

### 4.1 현재 구현 상태

| 항목 | 현재 상태 | 법적 요구 |
|------|----------|----------|
| 만 14세 미만 가입 제한 | ✅ 구현됨 | 필수 (아동 온라인 개인정보) |
| 법정대리인 동의 | ✅ 구현됨 | 필수 (미성년자) |
| 결제 제한 | ⚠️ 부분 구현 | 필수 (미성년자 결제) |
| 연령 제한 콘텐츠 게이트 | ⚠️ 없음 | 권장 |

### 4.2 미성년자 결제 한도

- **만 14세 미만**: 회원가입 불가
- **만 14세 이상 ~ 만 19세 미만**: 월 결제 한도 70,000원, 법정대리인 동의 필수, 법정대리인 취소권 고지
- **법적 근거**: 민법 제5조, 전자상거래법 제17조의2

1. 타인에 대한 폭언, 비방, 혐오 표현

6. 사기/기만 행위
7. 저작권 침해

9. **DT를 이용한 사행행위**

1단계: 경고
2단계: 7일 이용 정지
3단계: 30일 이용 정지

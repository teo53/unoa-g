## 2. DT(디지털 토큰) 법적 리스크

### 2.1 선불전자지급수단 해당 여부

| 검토 항목 | UNO A DT | 선불전자지급수단 | 판단 |
|----------|----------|----------------|------|
| 금전적 가치 | ✅ | ✅ | 해당 가능 |
| 전자적 저장 | ✅ | ✅ | 해당 |
| 대가 지급 | ✅ (유료 구매) | ✅ | 해당 |
| 양도성 | ❌ (양도 불가) | ✅ | **미해당** |
| 범용성 | ❌ (서비스 내 전용) | 다양함 | 검토 필요 |

**결론**: 양도 불가능하고 서비스 내 전용이므로 선불전자지급수단에 **해당하지 않을 가능성** 높음. 단, 정확한 판단은 법률 검토 필요.

### 2.2 리스크 완화 조치

```markdown
## DT 리스크 완화 체크리스트

✅ 명시적으로 "현금 환불 불가" 고지
✅ "타인 양도 불가" 약관 명시
✅ "서비스 내에서만 사용" 명시
✅ 1년 이상 미사용 시 소멸 규정 (선택)
✅ 충전금 별도 계정 관리 (신탁 또는 별도 계좌)
□ 분기별 DT 발행/소진 현황 내부 기록
□ 회계 처리 방침 확정 (부채 인식 시점)
```

### 2.3 DT 약관 필수 고지 사항

```markdown
## DT 이용 안내

1. DT는 UNO A 서비스 내에서만 사용 가능한 결제수단입니다.
2. DT는 현금으로 환불되지 않습니다.
3. DT는 타인에게 양도/판매할 수 없습니다.
4. 미사용 DT는 구매일로부터 5년간 유효합니다.
5. 회원 탈퇴 시 잔여 DT는 소멸됩니다.
6. DT 환불은 관련 법령에 따라 제한될 수 있습니다.
```

## 3. 크리에이터 정산 세무

### 3.1 크리에이터 유형별 처리

| 유형 | 세금 처리 | 필요 서류 | 현재 상태 |
|------|----------|----------|----------|
| 개인 (비사업자) | 기타소득 원천징수 (8.8%) | 주민등록번호 | ⚠️ 수집만 |
| 개인사업자 | 사업소득, 세금계산서 | 사업자등록번호 | ⚠️ 수집만 |
| 법인사업자 | 법인소득, 세금계산서 | 사업자등록번호, 법인등록번호 | ❌ |

### 3.2 원천징수 체크리스트

```markdown
## 원천징수 처리 체크리스트

### 비사업자 개인 크리에이터
□ 주민등록번호 수집 (암호화 저장)
□ 기타소득 8.8% (소득세 8% + 지방소득세 0.8%) 원천징수
□ 매월 원천세 신고/납부 (익월 10일까지)
□ 연간 지급명세서 제출 (익년 3월)
□ 크리에이터에게 원천징수영수증 발급

### 사업자 크리에이터
□ 사업자등록번호 확인 (국세청 API)
□ 세금계산서 수취 또는 발행
□ 부가세 처리 (수수료 매출세금계산서)
```

### 3.3 정산서 필수 항목

```typescript
interface PayoutStatement {
  // 기본 정보
  statementNumber: string;      // 정산서 번호
  periodStart: Date;            // 정산 기간 시작
  periodEnd: Date;              // 정산 기간 종료
  issueDate: Date;              // 발급일

  // 크리에이터 정보
  creatorName: string;
  creatorType: 'individual' | 'business';
  taxId: string;                // 주민번호 또는 사업자번호 (마스킹)

  // 수익 내역
  grossRevenue: number;         // 총 수익
  platformFee: number;          // 플랫폼 수수료 (20%)
  netRevenue: number;           // 순 수익

  // 세금
  withholdingTax?: number;      // 원천징수세 (비사업자)
  vat?: number;                 // 부가세 (사업자)

  // 실지급액
  payoutAmount: number;

  // 지급 정보
  bankName: string;
  accountNumber: string;        // 마스킹 (****1234)
  accountHolder: string;
}
```

## 9. 법률 자문 필요 사항

> 아래 항목은 반드시 법률 전문가 검토가 필요합니다.

1. **DT의 법적 성격**: 선불전자지급수단 해당 여부 최종 판단

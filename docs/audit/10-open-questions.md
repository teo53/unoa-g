## 4. 법무/컴플라이언스 질문

### Q12. DT 법적 분류 ✅ 해결됨
**질문**: ~~DT가 전자금융거래법상 선불전자지급수단에 해당하는가?~~

**결론 (2026-02-16 확정)**:
- DT는 **서비스 전용 디지털 이용권**이며, 선불전자지급수단에 **해당하지 않음**
- 근거: ① 양도 불가, ② 서비스 내 전용, ③ 현금화/환전 불가, ④ 제3자 결제수단 아님
- 적용 법체계: **전자상거래법** + **콘텐츠산업진흥법** (전자금융거래법 비해당)
- 회사 지위: **통신판매업자(총매법/Principal)** — 직접 판매자

**완료 조치**:
- [x] 법률 자문 기반 분석 완료 (`docs/legal/compliance/2026-02-16-comprehensive-compliance-report.md`)
- [x] 약관에 DT 법적 성격 명시 (`docs/legal/policies/02-dt-usage-terms.md`)
- [x] UI 전면 정비 — "충전" → "구매", "선불전자지급수단" → "디지털 이용권"

**결정 시점**: 2026-02-16 확정
**담당자**: Legal ✅
